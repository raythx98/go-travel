## Summary

Phase 1 is complete. This log covers fixes applied in the final session after the core Phase 1 implementation was done.

## Fixes Applied This Session

### 1. Gemini thought-signature Error 400
Gemini thinking models attach `ThoughtSignature []byte` to function call parts. Omitting it on replay causes Error 400. Fixed by storing the signature on `ContentBlock` and echoing it back in `toGeminiContents`. Also excluded `Part.Thought = true` parts from conversation history.

Files: `internal/llm/client.go`, `internal/llm/gemini.go`

### 2. Fallback broadened (ShouldFallback)
`IsRateLimitOrUnavailable` only triggered fallback on 429/â‰Ą500. Renamed to `ShouldFallback` and broadened to any `genai.APIError`. `context.Canceled` explicitly excluded. Router updated to call `ShouldFallback`.

Files: `internal/llm/gemini.go`, `internal/llm/router.go`

### 3. Open-Meteo 400 for past dates
The forecast API rejects dates in the past. The tool now uses `useArchive = daysAhead > 16 || daysAhead < 0`. Only far-future dates shift -1 year (`shiftDates`). Past dates use the archive API with actual dates (no shift). Date remapping in response only applies when `shiftDates` is true.

File: `internal/tools/weather.go`

### 4. SearXNG 403 (json format missing)
SearXNG returns 403 for formats not listed in `search.formats`. Added `json` to the list.

File: `searxng/settings.yml`

### 5. Redis not reachable from localhost
`docker-compose.yml` had no `ports:` on the Redis service, so it was only accessible within the Docker network. Added `ports: - "6379:6379"`.

File: `docker-compose.yml`

### 6. LangFuse crash-looping
`langfuse/langfuse:latest` pulled v3 which requires ClickHouse. Pinned to `langfuse/langfuse:2`. Added required `SALT` env var.

File: `docker-compose.yml`

### 7. Ollama port exposed + added to make dev
Added `ports: - "11434:11434"` to Ollama service and added `ollama` to `make dev` startup services.

Files: `docker-compose.yml`, `Makefile`
